## Табло теннисного матча.

Табло для теннисного матча, в котором можно создавать игроков, начислять им очки, просматривать список матчей с
пагинацией и фильтрами.

В проекте содержится файл import.sql, который сразу вставляет 21 участника (можно посмотреть на допустимые варианты имён)
и 14 матчей. Для проверки работы фильтров я рекомендую искать по комбинации 'char' (будет 7 матчей на 2 страницы).
Фильтр с частичным совпадением найдет все имена, содержащие 'char': напр. Charlotte и LatteChar.

### Деплой (до 29.03.2026):
[http://212.8.229.231:8080/TennisScoreboard/](http://212.8.229.231:8080/TennisScoreboard/)

По условиям [ТЗ](https://zhukovsd.github.io/java-backend-learning-course/projects/tennis-scoreboard/) правила упрощены:
* Матч играется до двух сетов (best of 3)
* При счёте 6/6 в сете, играется тай-брейк до 7 очков

### Требования для запуска:
* Java 17
* Apache Tomcat 11.0.18

### Для запуска:
1. Клонировать проект.
2. Собрать проект с расширением `.war`.
3. Развернуть `.war` в Tomcat.
4. Открыть в браузере [http://ВАШ_АЙПИ/TennisScoreboard/.]()
